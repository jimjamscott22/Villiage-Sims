# VillageSim Milestone 6 Implementation Plan

**Goal:** Clock (day/season/year) + speed controls; `crops.json` wheat planting/growth with watering and seasonal stall; crops render by stage.

**Branch:** `jimjamscott22/milestone-6-clock-crops-34b3`

**Design:** `docs/superpowers/specs/2026-07-20-milestone-6-design.md`

## Constraints

- Complete only M6 from `docs/villagesim-spec.md` and the M6 design doc.
- Keep M5 needs, jobs, Working FSM, and VillagerPanel.
- Do not add harvest, seed spending, weather, utility AI, or multi-villager.
- Speed scales tick interval only; pause still processes commands.

- Create: `src-tauri/src/sim/clock.rs`, `src-tauri/src/sim/crops.rs`, `src-tauri/data/crops.json`

## Tasks

1. **Clock** — `clock.rs` calendar + speed; wire into sim loop (pause / interval); unit tests for rollover; `advance_clock` command.
2. **Catalog + crops.json** — load wheat def; extend `Catalog` / `get_catalog`; crop kind index helper.
3. **Crops module** — plant validation, growth tick (season + water gates), `CropReady`, demolish cleanup; unit tests.
4. **World integration** — store crops; day-rollover water clear; TendCrops auto-plant at work-cycle start + water each Working tick; snapshot `clock` / `crops` / events.
5. **IPC** — `set_speed`, `plant_crop`, `advance_clock` (+ catalog crops).
6. **Frontend** — clock/speed chrome; plant mode; draw crops by stage; types + transport.
7. **Browser-demo parity** — clock, speed, plant, growth, watering, season stall, `advance_clock`.
8. **Verify** — `cargo test --lib`, `npm test`, `npm run build`; browser smoke (spring grow → winter stall); update `progress.md`; open PR.

## Acceptance

Wheat planted in spring visibly grows through stages (with TendCrops watering) and stalls after jumping to winter. Pause/1×/2×/3× work. No harvest required.
