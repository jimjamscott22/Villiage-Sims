# VillageSim Milestone 4 Implementation Plan

**Goal:** One villager pathfinds to a right-clicked tile and repaths when a building blocks the route.

**Branch pattern:** `jimjamscott22/<descriptive-name>-8d49`

**Tech add:** `pathfinding` crate on the Rust side.

## Constraints

- Complete only M4 from `docs/villagesim-spec.md` and the M4 design doc.
- Keep M3 placement/demolish and the command-channel architecture.
- Do not add needs, jobs, utility AI, crops, or clock UI.
- Cap A* expansion (~4000 nodes); cache paths; never repath every tick on failure.

## File map

- Create: `src-tauri/src/sim/pathfind.rs`, `src-tauri/src/sim/agents.rs`
- Modify: `src-tauri/src/sim/world.rs`, `commands.rs` (sim + Tauri), `mod.rs`, `Cargo.toml`, `snapshot.rs` (optional state byte)
- Modify: `src/render/Canvas.tsx`, `src/state/transport.ts`, `src/state/demoWorld.ts`, types as needed
- Update: `progress.md` when M4 demos

## Tasks

1. Add `pathfinding` dependency; implement A* helpers + unit tests (wall, no-path, diagonal corner cut forbidden).
2. Add `Villager` + `AgentState` FSM; spawn one walkable villager; tick movement along cached path (~2 tiles/s).
3. Wire `SimCommand::MoveVillagerTo` + Tauri `move_villager_to`; invalidate path on place/demolish occupancy changes.
4. Frontend: right-click → `moveVillagerTo`; browser-demo A*/BFS parity.
5. Tests + browser smoke (walk around a placed hut); update `progress.md`; open PR.

## Acceptance

Right-click a tile → villager walks there around water/buildings. Place a building on the path mid-walk → villager repaths around it (or Idles briefly and retries successfully).
